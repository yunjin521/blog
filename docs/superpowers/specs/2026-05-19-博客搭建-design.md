# 博客搭建方案设计

## 项目概览

- **目标**：零基础搭建个人博客，学习建站过程，做个人内容分享
- **预算**：首年 150 元，域名约 60 元/年
- **风格**：温暖文艺（奶油暖调 / 自然森系，待定）
- **内容**：图文混合，可能嵌入视频，不定期更新
- **关键约束**：国内访问优先，零代码起步

## 双线并行方案

两条线同时搭建，共用域名，中途任一方向更顺手就深耕那个方向。

### 方案 A — 低代码线（NotionNext + Cloudflare Pages）

- **写作入口**：Notion（可视化编辑器，零代码）
- **渲染引擎**：NotionNext（Notion 文章 → 博客网站）
- **部署平台**：Cloudflare Pages（替代 Vercel，国内访问更好）
- **适合场景**：快速出内容，不想折腾技术

### 方案 B — 可控线（Markdown 静态站 + Gitee Pages）

- **写作入口**：Markdown 文本文件（存放在项目文件夹中）
- **渲染引擎**：简单静态站点生成（HTML/CSS 模板）
- **部署平台**：Gitee Pages（国内直连，速度快）
- **适合场景**：内容完全自主可控，想学技术底层

## 内容结构

- **首页**：文章列表，带封面图，按时间排列
- **关于我**：个人介绍页
- **分类/标签**：按主题归类
- **友链**：互链其他博客

## 费用预算

| 项目 | 费用 |
|------|------|
| 域名（.com/.cn） | ~60 元/年 |
| Cloudflare DNS + CDN | 免费 |
| Cloudflare Pages / Gitee Pages | 免费 |
| Notion | 免费 |
| **合计** | **~60 元** |

## 升级路径

1. **阶段一（当前）**：免费托管 + 免费子域名，先上线
2. **阶段二**：绑自定义域名，独立品牌
3. **阶段三（可选）**：内容多了以后，可迁移到更复杂的技术栈（Hexo/Hugo 等），Notion 内容一键导出 Markdown 零丢失

## 实施步骤

1. 注册 Notion 账号，复制博客模板
2. 注册域名（阿里云/腾讯云/NameSilo）
3. 方案 A：部署 NotionNext 到 Cloudflare Pages
4. 方案 B：搭建 Markdown 静态站，部署到 Gitee Pages
5. DNS 配置：域名指向两个站点
6. 对比体验，选择主线继续深耕
