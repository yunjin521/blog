无人机学习路线图

目标：从画飞控板开始，掌握完整无人机技术栈（自主飞行 + ROS + 就业/竞赛级别）
起点：有Cinebot25 V2 + Vapor-X5两架飞机，会手动飞行，会Betaflight基础调参，有普中51单片机（闲置）
偏好：跟开源项目复刻，边做边学
预算：500-2000元 | 时间：每周5-10小时
总周期：8-14个月 | 总预算：~1200元

核心原则

先飞起来 → 拆解理解 → 复刻改进 → 自主设计

不能上来就画飞控板。飞控板是整个无人机系统的中枢，如果没理解它在真实系统中的角色，画出来的大概率是通电就烧的砖。你已经有飞行经验，这比纯理论学生强太多——你需要的是把"知道怎么飞"转化为"知道它为什么能飞"。

阶段零：电子基础 + 51单片机唤醒（3-4周，~100元）

电子理论（1周）

用你手头两架飞机的飞控板当实物教材：电阻分压、上拉/下拉、电容去耦/滤波、MOSFET作为开关管、LDO vs DC-DC（BEC降压）、看懂飞控供电链：电池 → BEC → 5V → 3.3V LDO → MCU/IMU

51单片机（2周）

按顺序跑通：GPIO输出（点灯）→ GPIO输入+外部中断（按键检测）→ 定时器/计数器（理解中断周期 = 飞控PID循环的基础）→ UART串口通信（printf到电脑 = 飞控debug手段）→ I2C通信（读传感器）→ ADC采样。

不需要学到很精。重点是理解：寄存器怎么操作、中断怎么响应、I2C/SPI/UART怎么工作——这三点是飞控固件的骨架。

焊接练习（同步）

买焊接练习板（~30元），先焊直插元件再焊0805贴片。工具：936焊台（~50元）+ 万用表DT9205（~20元）+ 焊锡丝 + 助焊剂。

阶段一：解剖现有飞控 + 过渡STM32（4-6周，~200元）

解剖飞控（2周）

拆开Cinebot25或Vapor-X5找飞控板：对照Betaflight开源原理图识别板上芯片，用万用表测量供电链，验证理解。

STM32入门（4周）

  1. 买STM32F405核心板（~50元）+ ST-Link V2调试器（~30元）
  2. 跑通外设：GPIO → 中断 → 定时器PWM → SPI读IMU → UART
  3. 买ICM-42688或MPU6050模块（~15元）
  4. 用STM32读IMU数据，从串口打印姿态角——飞控的第一步

阶段二：设计自己的飞控板（8-12周，~400元）

用立创EDA画出完整飞控原理图+Layout，打样、焊接、烧录、跑Betaflight。

PCB设计子模块

#模块关键要点
1供电链2-6S输入→5V BEC→3.3V LDO，注意电流和散热
2MCU最小系统STM32F405RGT6+晶振+复位+BOOT
3IMUICM-42688-P（SPI），远离发热和震动源
4OSDAT7456E，独立LDO供电
5接收机ELRS CRSF协议（UART）
6电调输出4路PWM，走线不贴电源线

Layout注意事项

  • IMU远离发热元件和安装孔（震动）
  • 电源走线避开信号线
  • 地平面尽量完整
  • DRC检查通过再打样

调试流程

  1. 上电→测各路电压（先不焊MCU）
  2. 焊MCU→ST-Link烧录Betaflight Bootloader
  3. Betaflight Configurator连接→传感器读数正常
  4. 装飞机实飞

阶段三：飞控算法 + 自主飞行（12-16周，~500元）

飞控算法（6周）

从零写简化版飞控固件：Mahony/Madgwick互补滤波姿态解算→串级PID（角速度内环+角度外环）→混控输出（四轴X型混控矩阵）→移植到自己飞控板实现自稳飞行。

自主飞行（6周）

加入GPS+磁力计模块→移植INAV的INS/EKF导航代码→实现定点悬停→ESP32数传+QGroundControl地面站。

ROS/仿真入门（4周）

ROS2 Humble+Gazebo环境搭建→PX4 SITL仿真→MAVROS/MAVSDK通信→选配树莓派/RK3588机载计算机跑视觉SLAM。
